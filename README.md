# Golang Practice

A gallery of simple implementations of common tasks and tools using Golang. These come from practice exercises while learning the language, and the implementations are far from perfect.

## Add New Item

```bash
mkdir xgrep
cd xgrep
go mod init xgrep
touch main.go
cd ..
go work use xgrep
```
